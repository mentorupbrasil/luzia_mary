# Aplicar o redesign premium

Este pacote é a versão de referência do redesign editorial da plataforma Luzia Mary.

## Aplicação recomendada

1. Faça backup do repositório atual.
2. Preserve o arquivo `.env.local` e as variáveis configuradas na Vercel.
3. Substitua os arquivos do projeto pelos arquivos deste pacote.
4. Execute:

```bash
npm install
npm run typecheck
npm run lint
npm run build
```

5. Faça commit em uma branch de preview antes de publicar em produção.

## Conteúdo que ainda precisa ser fornecido pela campanha

- fotografia oficial em alta resolução, preferencialmente recortada e com fundo transparente;
- links oficiais das redes sociais;
- número oficial de WhatsApp;
- e-mail confirmado;
- número de urna, CNPJ e responsável legal após o registro;
- biografia e propostas revisadas pela candidata e pelo jurídico eleitoral.

A estrutura visual e funcional já está preparada para receber esses dados sem nova reconstrução.
